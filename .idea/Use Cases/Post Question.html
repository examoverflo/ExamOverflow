Post Question - Casual Use Case

A Student wants to post a qusetion on the system. The Student is logged into the system and has navigated to the question page. The Student types his question into the provided text box. The Student also selects a Module and Topic for his question. The system checks to make sure the question contains legal characters and has not been asked by another student. The System will assign an ID to the question. The System displays a confiramtion message for the Students. Possible Erros:

  • The Student has failed to give a Module or Topic for the question, the System displays an error message
  • The question contains illegal characters, the System display an error message
  • The same question has already been asked by another Student, the System dislpays a message
